factorial

Условие
Факториалом числа n называется произведение 1 × 2 × ... × n. Обозначение: n!.
По данному натуральному n вычислите значение n!.


```
from functools import reduce
number = int(input())
res = reduce(lambda x,y: x*y, range(1,number+1))
print(res)
```
